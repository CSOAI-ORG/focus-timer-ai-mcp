# Focus Timer AI MCP

Run deep-focus timers with distraction logging and daily focus score calculations.

## Tools

- `start_focus` — Start a focus session.
- `log_distraction` — Log a distraction event.
- `end_focus` — End the current focus session and score it.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m focus-timer-ai-mcp.server
```
